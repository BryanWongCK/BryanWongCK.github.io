# Projects Media Folder

This folder is for storing screenshots and GIFs that showcase your portfolio projects.

## How to Use

1. **Add your project media files** to this folder
2. **Name your files** descriptively (e.g., `project1-screenshot.png`, `project2-demo.gif`)
3. **Update index.html** to reference your media files

## Supported Formats

- **Screenshots**: PNG, JPG, JPEG, WebP
- **Animated Demos**: GIF, MP4 (as video)

## Recommended Dimensions

- **Desktop Screenshots**: 1920x1080 or 1280x720
- **Mobile Screenshots**: 375x667 or 414x896
- **GIFs**: Keep under 5MB for fast loading

## File Naming Convention

Use descriptive names that indicate the project and media type:
- `project-name-screenshot.png`
- `project-name-demo.gif`
- `project-name-mobile.png`

## Example Structure

```
projects/
├── project1-screenshot.png
├── project1-demo.gif
├── project2-screenshot.png
├── project2-demo.gif
├── project3-screenshot.png
├── project4-demo.gif
└── README.md (this file)
```

## Tips

1. **Optimize your images** before uploading to reduce file size
2. **Use descriptive alt text** in your HTML for accessibility
3. **Keep GIFs short** (3-10 seconds) for better performance
4. **Consider using WebP** format for better compression
5. **Test on mobile devices** to ensure images load quickly

## Adding a New Project

To add a new project to your portfolio:

1. Add your screenshot/GIF to this folder
2. Open `index.html`
3. Copy an existing project card
4. Update the image source, title, description, and tags
5. Commit and push your changes

Your portfolio will automatically update on GitHub Pages!
